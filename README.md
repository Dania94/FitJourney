# FitJourney

FitJourney is a full-stack fitness companion for planning nutrition and workouts while tracking long-term body progress.

## Project status

The project is currently in **Sprint 0: Foundations and Discovery**. During this sprint, the product vision, MVP scope, architecture, and domain model will be defined before implementation begins.

## Planned capabilities

- Personalized nutrition and meal planning
- Calorie and macronutrient tracking
- Cuisine and diet preferences
- Monthly workout planning and workout logs
- Weight and body-measurement tracking
- Progress visualization

## Planned technology stack

- Java 21 and Spring Boot 3
- Angular and TypeScript
- PostgreSQL
- Docker

The technology choices will be recorded formally as part of the Sprint 0 architecture story.

## Documentation

Project documentation will be added under `docs/` as each Sprint 0 story is completed.
